# AI QA Testing Project

## Project Overview

This project demonstrates Manual Testing, Automation Testing, and AI Testing on a real-world web application.

## Objectives

* Perform Manual Testing on web application features.
* Automate test scenarios using Python and Selenium.
* Validate AI responses through AI Testing techniques.
* Document test cases, bug reports, and test execution results.

## Tools Used

* Python
* Selenium WebDriver
* Pytest
* Git & GitHub
* ChatGPT
* Excel
